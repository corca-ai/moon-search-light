# Moon Search Light - Product Specification

**AI 기반 학술 논문 검색 및 연구 아이디어 도출 플랫폼**

---

## 1. 개요

### 1.1 목적
학술 연구자들이 논문을 검색하고, AI 분석을 통해 파악하며, 후속 연구 아이디어를 도출하는 웹 애플리케이션

### 1.2 핵심 가치
- **검색**: Semantic Scholar 2억 건 이상 논문 검색
- **스마트 정렬**: 최신 연구 우선 + 인용수 반영
- **AI 요약**: 구조화된 한국어 논문 요약
- **관심 주제 분석**: 선택/제외 패턴 기반 자동 분석
- **관련도 표시**: 선택한 논문과 비슷한 주제의 논문 자동 표시
- **Research Assistant**: AI 대화를 통한 후속 연구 아이디어 도출
- **시각적 프리뷰**: ArXiv 논문 스냅샷 이미지

### 1.3 대상 사용자
학술 연구자, 대학원생, 후속 연구 기획자

---

## 2. 화면 구성

### 2.1 레이아웃
- **좌측**: 사이드바 (연구 노트 목록, 접힘/펼침)
- **우측**: 메인 콘텐츠 (선택 영역 + 검색/Assistant)

### 2.2 사이드바
- 로고 + 타이틀
- 연구 노트 목록 (이름, 논문 수, 수정일)
- 노트 생성/이름 변경/삭제
- 접힘/펼침 토글

### 2.3 메인 콘텐츠
- **선택 영역**: 선택된/제외된 논문, 관심 주제 요약
- **탭 패널**: 검색/연구 탭으로 모드 전환
  - 검색 탭: 검색창 + 결과 목록
  - 연구 탭: Research Assistant 채팅 (1개 이상 선택 시 활성화)
- **헤더**: 연구 개요 다운로드 (연구 탭에서 대화 진행 시)

---

## 3. 검색 기능

- **데이터 출처**: Semantic Scholar API
- **검색**: 최대 100개, 초기 20개 표시, "더 보기"로 추가 로드
- **정렬**: 추천순(기본), 관련성, 최신순, 인용순
- **추천순**: 연도 가중치(60%) + 인용수 가중치(40%)

---

## 4. 논문 카드

### 검색 결과 카드
- 썸네일, 제목, 연도, 인용수, 논문 링크
- AI 분석 결과 (개요, 목표, 방법론, 결과)
- 키워드 태그, 초록 번역

### 선택된/제외된 카드
- 간결한 형태, 클릭 시 상세 모달

### 액션
- ⭐ 선택, ✕ 제외, 복원

---

## 5. AI 기능

### AI 요약
- 구조: 개요, 목표, 방법론, 결과
- 배치 처리: 3개씩 순차
- 키워드: 3-5개 추출

### 관심 주제 요약
- 선택/제외 시 자동 분석 (500ms 디바운스)
- 명사형 종결, 100자 이하

### 초록 번역
- GPT로 한국어 번역

### 관련도 표시
- 선택한 논문과 후보 논문의 주제 연관성 측정
- 제목 + 초록 기반 AI 분석
- 0-100% 스케일로 관련도 표시 (높을수록 비슷한 주제)

---

## 6. Research Assistant

### 활성화
- 1개 이상 논문 선택 필요
- 2개 이상 시 통합 컨텍스트 자동 생성

### 기능
- 마크다운 렌더링 채팅
- 논문 비교 분석, Research Gap 식별
- 후속 연구 아이디어 제안

### 맥락 변경 감지
- 논문 추가/제거 시 이전 대화 유지
- 변경 사항 자동 알림 (추가/제거된 논문 표시)
- 새로운 맥락으로 통합 분석 자동 갱신

### 연구 개요 다운로드
- 대화 내용 마크다운 파일로 저장

---

## 7. 외부 연동

- **Moonlight**: ArXiv 논문 뷰어 연동
- **Semantic Scholar**: 논문 상세 페이지 링크
