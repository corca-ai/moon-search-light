# Moon Search Light - Product Specification

**AI-Powered Academic Paper Search & Recommendation System**

Semantic Scholar API와 OpenAI를 활용한 학술 논문 검색, 분석, 추천 플랫폼

---

## 1. 개요

### 1.1 목적
학술 연구자들이 효율적으로 관련 논문을 검색하고, AI 기반 요약을 통해 빠르게 논문을 파악하며, 의미론적 유사도 기반 추천을 받을 수 있는 웹 애플리케이션

### 1.2 핵심 가치
- **효율적인 검색**: Positive/Negative 키워드 기반 정밀 검색
- **AI 요약**: 구조화된 형식의 한국어 논문 요약
- **스마트 추천**: 벡터 임베딩 기반 의미론적 유사 논문 추천
- **시각적 프리뷰**: ArXiv 논문 스냅샷 이미지 제공

### 1.3 대상 사용자
- 학술 연구자
- 대학원생
- 논문 리뷰어
- 문헌 조사를 수행하는 연구자

---

## 2. 검색 기능

### 2.1 키워드 검색 방식

**Positive Keywords (포함 키워드)**
- 검색 결과에 반드시 포함되어야 하는 키워드
- 공백으로 구분하여 여러 키워드 입력 가능
- 예: `graph attention networks` → 세 단어가 모두 관련된 논문 검색

**Negative Keywords (제외 키워드)**
- 검색 결과에서 제외할 키워드
- 제목 또는 초록에 해당 키워드가 포함된 논문은 "Excluded" 영역으로 분류
- 예: `survey review` → 서베이/리뷰 논문 제외

### 2.2 검색 범위 및 표시
- Semantic Scholar에서 최대 100개 논문 검색
- 화면에는 상위 20개 논문 표시
- 나머지 논문은 추천 시스템에서 활용

### 2.3 검색 결과 정렬 (Smart Sorting)
검색 결과는 다음 우선순위로 자동 정렬됩니다:

1. **제목 정확히 일치**: 검색어와 논문 제목이 정확히 일치하는 경우 최상단
2. **제목에 검색어 포함**: 제목에 검색어가 포함된 논문 우선
3. **API 관련성 순서**: Semantic Scholar가 판단한 관련성 순서 유지

**예시**:
- 검색어: `graph attention networks`
- 결과: "Graph Attention Networks" 논문이 최상단에 표시

### 2.4 추가 정렬 옵션
사용자가 선택할 수 있는 정렬 옵션:
- **관련성**: 기본 정렬 (Smart Sorting 적용)
- **최신순**: 발행 연도 내림차순
- **오래된순**: 발행 연도 오름차순
- **인용 많은순**: 인용 수 내림차순

## 3. 화면 구성

### 3.1 Three-Panel 시스템

**Left Panel - Selected Papers (상단)**
- 사용자가 "읽기로 확정한" 논문 목록
- Candidate에서 ↑ 버튼으로 이동
- AI 요약 및 스냅샷 이미지 표시
- 이 목록을 기반으로 유사 논문 추천이 작동

**Left Panel - Candidate Papers (하단)**
- 추천 후보 논문 목록
- 정렬 옵션 선택 가능
- ↑ 버튼으로 Selected로 이동 가능
- Selected Papers가 변경되면 자동으로 재정렬됨

**Right Panel - Excluded Papers**
- Negative 키워드로 제외된 논문
- 초록 미리보기 제공 (3줄)
- 인용순 정렬

### 3.2 논문 카드 정보
각 논문 카드에 표시되는 정보:
- 논문 제목
- 발행 연도
- 인용 수
- 저자 목록
- AI 분석 결과 (Selected/Candidate만)
- 스냅샷 이미지 (있는 경우)
- 논문 보기 링크

---

## 4. AI 요약 기능

### 4.1 요약 구조
각 논문에 대해 다음 4가지 항목을 한국어로 요약합니다:

| 항목 | 설명 |
|------|------|
| **개요** | 논문의 전체적인 내용 요약 (1-2문장) |
| **연구 목표** | 논문이 해결하고자 하는 문제 (1-2문장) |
| **방법론** | 문제 해결을 위한 접근 방식 (1-2문장) |
| **결과** | 주요 결과 및 기여점 (1-2문장) |

### 4.2 키워드 추출
- 논문당 3-5개의 핵심 키워드 추출 (영어)
- 키워드 클릭 시 검색창에 자동 추가
- 이미 검색창에 있는 키워드는 중복 추가되지 않음

### 4.3 요약 대상
- Selected Papers와 Candidate Papers에만 AI 요약 제공
- Excluded Papers는 초록 미리보기만 제공
- 초록이 없는 논문은 요약 불가

## 5. 추천 시스템

### 5.1 작동 원리
사용자가 Selected Papers에 논문을 추가하면, 시스템이 자동으로 유사한 논문을 찾아 Candidate Papers를 재정렬합니다.

### 5.2 유사도 계산 방식

**의미론적 유사도 (60%)**
- 논문의 제목과 초록을 벡터로 변환
- Selected Papers의 평균 벡터와 각 논문의 벡터 간 유사도 계산
- 내용이 비슷할수록 높은 점수

**인용 점수 (40%)**
- 인용 수가 많은 논문에 가산점 부여
- 로그 스케일로 정규화하여 극단적인 차이 완화
- 검증된 논문을 우선 추천

### 5.3 추천 동작
- Selected Papers가 비어있으면: 기본 검색 순서 유지
- Selected Papers에 논문 추가 시: 즉시 Candidate Papers 재정렬
- Selected Papers에서 논문 제거 시: 다시 재정렬

### 5.4 추천 범위
- 검색된 100개 논문 중 Selected에 없는 논문 대상
- 상위 20개만 Candidate Papers에 표시

## 6. 이미지 스냅샷

### 6.1 데이터 출처
- ArXiv 논문의 주요 페이지 캡처 이미지
- 약 22만 개 논문의 스냅샷 보유

### 6.2 표시 방식
- 논문당 최대 3개 썸네일 표시
- 클릭 시 전체 화면 모달로 확대
- 모달 외부 클릭 또는 X 버튼으로 닫기

### 6.3 매칭 방식
- 논문 제목을 기준으로 자동 매칭
- 대소문자, 특수문자 무시하고 매칭
- 매칭되는 스냅샷이 없으면 표시하지 않음

---

## 7. 외부 링크

### 7.1 Moonlight 연동
ArXiv PDF URL이 있는 논문의 경우:
- "Moonlight에서 보기 →" 링크 표시
- [themoonlight.io](https://www.themoonlight.io)에서 논문 열람

### 7.2 기본 링크
PDF URL이 없는 논문의 경우:
- "논문 보기 →" 링크 표시
- Semantic Scholar 페이지로 이동
