# Moon Search Light - Product Specification

**AI 기반 학술 논문 검색 및 연구 아이디어 도출 플랫폼**

---

## 1. 개요

### 1.1 목적
학술 연구자들이 논문을 검색하고, AI 분석을 통해 파악하며, 후속 연구 아이디어를 도출하는 웹 애플리케이션

### 1.2 핵심 가치
- **검색**: Semantic Scholar 2억 건 이상 논문 검색
- **스마트 정렬**: 최신 연구 우선 + 인용수 반영
- **AI 요약**: 구조화된 한국어 논문 요약
- **관심 주제 분석**: 선택한 논문 기반 자동 분석
- **관련도 표시**: 선택한 논문과 비슷한 주제의 논문 자동 표시
- **Research Assistant**: AI 대화를 통한 후속 연구 아이디어 도출
- **시각적 프리뷰**: ArXiv 논문 스냅샷 이미지

### 1.3 대상 사용자
학술 연구자, 대학원생, 후속 연구 기획자

---

## 2. 화면 구성

### 2.1 레이아웃 (3단 구조)
- **좌측**: 연구 노트 사이드바 (접힘/펼침)
- **중앙**: 메인 콘텐츠 (검색 영역)
- **우측**: 연구 패널 (접힘/펼침)

### 2.2 좌측 사이드바
- 로고 + 타이틀
- 연구 노트 목록 (이름, 논문 수, 수정일)
- 노트 생성 (자동 번호 부여: 새 연구, 새 연구 2, ...)
- 노트 이름 변경/삭제
- 접힘/펼침 토글

### 2.3 메인 콘텐츠
- **헤더**: 세션 타이틀 (클릭하여 편집 가능), 다운로드 버튼
- **선택 영역**: 선택된/검토 완료 논문, 관심 주제 요약
- **검색 영역**: 검색창 + 결과 목록 (항상 표시)

### 2.4 우측 연구 패널
- **펼침 상태**: 560px 너비
  - 통합 컨텍스트 분석 (리로드 버튼)
  - Research Assistant 채팅
- **접힘 상태**: 48px 너비 바
  - 아이콘 + "Research" 텍스트
  - 메시지 개수 배지

---

## 3. 검색 기능

- **데이터 출처**: Semantic Scholar API
- **검색**: 최대 100개, 초기 20개 표시, "더 보기"로 추가 로드
- **정렬**: 추천순(기본), 관련성, 최신순, 인용순
- **추천순**: 연도 가중치(60%) + 인용수 가중치(40%)

---

## 4. 논문 카드

### 검색 결과 카드
- 썸네일, 제목, 연도, 인용수, 논문 링크
- AI 분석 결과 (개요, 목표, 방법론, 결과)
- 키워드 태그, 초록 번역

### 선택된/검토 완료 카드
- 간결한 형태, 클릭 시 상세 모달

### 액션
- ⭐ 선택, ✓ 검토 완료, 다시 검토

---

## 5. AI 기능

### AI 요약
- 구조: 개요, 목표, 방법론, 결과
- 배치 처리: 3개씩 순차
- 키워드: 3-5개 추출

### 관심 주제 요약
- 선택 시 자동 분석 (500ms 디바운스)
- 명사형 종결, 100자 이하

### 초록 번역
- GPT로 한국어 번역

### 관련도 표시
- 선택한 논문과 후보 논문의 주제 연관성 측정
- 제목 + 초록 기반 AI 분석
- 0-100% 스케일로 관련도 표시 (높을수록 비슷한 주제)

---

## 6. Research Assistant

### 우측 패널 구조
- 검색 영역과 동시 표시 가능
- 접기/펼치기로 화면 공간 조절
- 펼침 시 검색 영역 자동 축소

### 통합 컨텍스트 분석
- 2개 이상 논문 선택 시 활성화
- "분석" 버튼으로 수동 실행/리로드
- 분석 결과: 공통 문제, 공통 방법론, 주요 차이점, 연구 지형
- 세션에 저장되어 복원됨

### 채팅 기능
- 마크다운 렌더링
- 논문 비교 분석, Research Gap 식별
- 후속 연구 아이디어 제안

### 연구 개요 다운로드
- 대화 내용 마크다운 파일로 저장

---
